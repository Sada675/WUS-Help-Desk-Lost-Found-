Frequently Asked Questions

Find answers about how WUS Lost & Found system works

How does the Lost & Found system work?
Users can report lost or found items through the system. Admin verifies and matches reports. Once matched, users are contacted and items are returned safely.
What should I do if I lost something?
Go to "Report Lost" page, fill in item details like description, location, and time. The system will try to match it with found items.
How do I report a found item?
If you find any item on campus, submit it using "Report Found". Admin will verify and connect it with the rightful owner.
How will I get my item back?
Once a match is found, admin will contact you via your registered details like phone or email.
Is my data safe?
Yes, all user data is securely stored and only used for lost & found matching purposes.
